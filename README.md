# 🚗 Car Rent, Car Repair & Service Management System

An ASP.NET Core MVC full-stack web application developed as an ITI Graduation Project.  
The system provides car rental services, car repair booking, and service management with a secure and scalable architecture.

---

## 📌 Project Overview

This system connects three main roles:
- 👤 Users: Rent cars and book repair services
- 🔧 Mechanics: Manage repair requests and service schedules
- 🛠️ Admins: Control system data, users, cars, and services

The platform ensures a seamless experience for booking, payments, and communication between users and service providers.

---

## ✨ Features

### 🔐 Authentication & Authorization
- Role-Based Access Control (RBAC)
- Secure login & registration system
- Separate dashboards for Admin, Mechanic, and User

### 🚗 Car Rental System
- Browse available cars
- Book and manage rentals
- Check availability in real time

### 🔧 Car Repair & Services
- Request repair services
- Assign mechanics to requests
- Track service status

### 💳 Payment Integration
- PayPal payment gateway integration for secure transactions

### 🤖 AI Chatbot
- Integrated Google Gemini chatbot for user assistance

### 📧 Notifications System
- Automated email notifications using SMTP

---

## 🏗️ Architecture & Design
- N-Tier Architecture
- ASP.NET Core MVC architecture
- Clean separation of concerns
- Scalable and maintainable structure
